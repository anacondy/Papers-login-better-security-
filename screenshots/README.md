# Screenshots

This directory contains screenshots of the application for documentation purposes.

## Required Screenshots

To complete the documentation, please add the following screenshots:

1. **desktop-view.png** - Full desktop view of the terminal interface
2. **mobile-view.png** - Mobile view showing responsive design (16:9 or 20:9 aspect ratio)
3. **search-modal.png** - Search modal opened with Ctrl+K

## How to Add Screenshots

1. Run the application locally:
   ```bash
   python app.py
   ```

2. Take screenshots:
   - Desktop: Use browser at 1920x1080 resolution
   - Mobile: Use browser DevTools with mobile emulation (e.g., iPhone 12 or Galaxy S20)
   - Search: Press Ctrl+K to open search modal

3. Save images in this directory with the names above

4. Commit and push to update documentation

## Guidelines

- Use PNG format for best quality
- Desktop screenshots: 1920x1080 or 1280x720
- Mobile screenshots: Actual device resolution (e.g., 390x844 for iPhone 12)
- Keep file sizes reasonable (< 500KB per image)
- Ensure screenshots show actual working features
